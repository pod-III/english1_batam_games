QUICK GUIDE

1. ADD LAYERS

Use the BOTTOM DOCK to add new modules like Vocab lists, Rules, or Images.

2. EDIT CONTENT

CLICK any block on the poster to open the editor. Change text, colors, and sizes instantly.

3. ARRANGE

Inside the editor, use BACK and FWD to change layer order. Use COPY to duplicate.

4. SAVE & SHARE

Your work auto-saves locally. Use CTRL + S to force save. Export as JSON to backup.
